【快速开始】

1. 双击「一键安装.bat」 → 等待安装完成
2. 双击「一键启动.bat」 → 浏览器自动打开
3. 按照引导向导完成配置
4. 输入文章主题，点击生成

【需要准备】

- DeepSeek API Key (免费注册: https://platform.deepseek.com)
- 微信公众号 (用于发布)

【文档】

详细说明请查看 docs/ 文件夹：
- 快速开始.md → 10分钟安装教程
- 故障排查.md → 常见问题解决

【系统要求】

- Windows 10/11
- Python 3.12+ (安装包会自动检查)

遇到问题请查看 logs/ 目录下的日志文件。
