# 公众号AI智能运营系统 V2.2

## 快速开始（3步）

### 第1步：安装环境
双击 **「一键安装.bat」**，等待完成。

### 第2步：配置密钥
1. 打开 pp\.env.example，另存为 pp\.env
2. 填入你的 DeepSeek API Key
3. 可选：填入火山方舟 API Key（生图用）

### 第3步：启动系统
双击 **「一键启动.bat」**，浏览器自动打开。
