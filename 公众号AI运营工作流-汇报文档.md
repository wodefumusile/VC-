# 公众号AI自动运营工作流
## 项目汇报

---

### 一、项目概述

基于 **n8n 自动化平台 + DeepSeek 大模型 + 微信服务号API**，搭建了一套完整的公众号文章自动运营流水线。**输入一句话主题，输出一篇可直接发布的公众号文章（草稿）**，全程无需人工干预。

---

### 二、核心能力

| 功能 | 说明 |
|------|------|
| 🤖 一句话生成文章 | 输入主题/标题，AI 自动完成全部创作 |
| 📋 智能结构规划 | AI 分析行业趋势+用户画像，规划 5-8 章结构 |
| ✍️ AI 内容生成 | DeepSeek 大模型撰写正文 |
| 🔍 SEO 优化 | 自动提取关键词，优化搜索排名 |
| 🎨 图文排版 | 自动 Markdown 转公众号 HTML |
| 🛡️ 内容质检 | AI 评分 + 敏感词检测双重保障 |
| 📤 一键草稿/发布 | 自动存草稿，支持审核后发布或全自动发布 |
| 📚 品牌知识库 | 企业资料导入，保证内容贴合品牌调性 |
| ⏰ 定时运行 | 可设每日 9 点自动选题+创作 |

---

### 三、工作流程

`
输入主题
  ↓
品牌知识库注入 → Prompt工程增强
  ↓
DeepSeek AI ① 行业分析 + 用户画像 + 结构规划（5-8章）
  ↓
DeepSeek AI ② 正文生成（2000-4000字）
  ↓
DeepSeek AI ③ SEO 关键词优化
  ↓
DeepSeek AI ④ 图文排版建议
  ↓
DeepSeek AI ⑤ 内容质量检查（评分 + 问题 + 修改建议）
  ↓
平台违规检测（敏感词扫描）
  ↓
Markdown → 公众号HTML（自动排版）
  ↓
保存微信草稿 ← 或 → 自动发布
  ↓
通知推送
`

---

### 四、技术架构

| 组件 | 选型 | 说明 |
|------|------|------|
| 自动化引擎 | n8n（Docker本地部署） | 开源、可视化、32个节点 |
| AI 模型 | DeepSeek-Chat | 性价比高，中文能力强 |
| 内容安全 | wordscheck（本地） | 敏感词检测，数据不外传 |
| 微信接口 | 服务号 API | 草稿管理 + 发布 |
| 部署方式 | Docker 本地 | 零服务器成本，自己电脑跑 |

### 六、使用方式

**手动模式（当前）**：访问 n8n 后台 → 输入主题 → 自动生成草稿 → 人工审核 → 手动发布

**自动模式**：设定每日 9 点 → 自动选题 → 自动生成 → 自动发布（需开启）
