# 公众号AI运营系统 — 用户手册 v2.1.3

## 系统介绍

本系统帮助你自动生成微信公众号文章：输入主题 → AI 写文章 → 自动配图(可选) → 专业排版 → 一键发布。

支持两种运行模式：
- AI 完整模式：AI 写文章 + AI 配图（需配置图片 API Key）
- 纯文本模式：AI 写文章，无配图（只需 DeepSeek Key）

## 第一次使用

1. 在浏览器打开系统地址（默认 http://localhost:8000）
2. 跟随屏幕上的新手引导完成 4 步配置
3. 或点击左侧菜单「配置」手动设置

## 配置 API Key

### DeepSeek AI Key（必填）

1. 前往 platform.deepseek.com 注册
2. 获取 API Key（以 sk- 开头）
3. 在配置页填入，点击「测试连接」
4. 看到绿色状态即为配置成功

### 图片生成 Key（可选）

1. 如需自动配图，前往即梦 AI 获取 Key
2. 填入配置页，点击测试
3. 不填则自动进入纯文本模式

## 配置知识库（可选）

如果你使用 Obsidian 做笔记：

1. 在配置页填入 Vault 文件夹的完整路径
2. 例如：C:\\Users\\你的用户名\\Documents\\Obsidian\\MyVault
3. 点击「同步知识库」确认连接成功
4. AI 生成文章时会自动引用你的笔记内容

## 生成第一篇文章

1. 点击「文章生成」
2. 在输入框填入主题，如「2024年AI行业趋势」
3. 选择风格和长度
4. 点击「一键生成并发布」
5. 等待 30-90 秒
6. 点击「查看正文」预览完整文章
7. 点击「发布到微信」保存为公众号草稿

## 查看历史

点击「任务记录」可查看所有生成过的文章，点击「查看」重新查看文章内容。

## 发布到微信

系统自动将文章保存为微信公众号草稿，你可以在微信公众平台后台找到并群发。

首次发布需要手动扫码登录微信公众平台。

## 模块开关说明

| 开关 | 开启 | 关闭 |
|------|------|------|
| AI 生成 | 正常写文章 | 无法生成 |
| 知识库增强 | AI 引用你的笔记 | 纯 AI 创作 |
| 图片生成 | AI 自动配图 | 纯文本文章 |
