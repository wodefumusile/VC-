# 微信公众号AI自动运营系统 安装教程

> 适用版本：v1.0 | 适用对象：普通电脑用户（无需编程知识）

---

## 第一章 使用前准备

### 电脑要求

- 操作系统：Windows 10 或 Windows 11（64位）
- 内存：建议 8GB 以上
- 硬盘：至少 10GB 可用空间
- 网络：能正常上网

### 需要准备的账号

| 序号 | 准备事项 | 说明 |
|------|----------|------|
| 1 | 微信公众号账号 | 已注册的个人订阅号，知道账号密码 |
| 2 | DeepSeek API Key | AI 生成文章的密钥，获取方法见第四章 |
| 3 | 手机微信 | 首次登录需要扫码 |

## 第二章 安装必要软件

### 1. 安装 Python

**为什么要装 Python？**

本系统是用 Python 语言编写的，需要 Python 来运行。

**安装步骤：**

① 打开浏览器，访问 https://www.python.org/downloads/

② 点击黄色大按钮 **Download Python 3.12.x** 下载安装包

③ 双击下载的安装包（`python-3.12.x-amd64.exe`）

④ **重要！** 在安装界面最下方，勾选 **"Add Python to PATH"**（添加 Python 到系统路径）

⑤ 点击 **Install Now**（立即安装）

⑥ 等待安装完成，点击 **Close**（关闭）

⑦ 验证安装：按键盘 **Win + R**，输入 `cmd` 回车，在黑色窗口中输入：
```
python --version
```
如果显示类似 `Python 3.12.x`，说明安装成功。

### 2. 安装 Docker Desktop

**为什么要装 Docker？**

Docker 用来运行一个叫 n8n 的工具，帮助我们编排整个自动化流程。

**安装步骤：**

① 打开浏览器，访问 https://www.docker.com/products/docker-desktop/

② 点击 **Download Docker Desktop for Windows** 下载

③ 双击下载的安装包，按照提示一步步安装

④ 安装完成后，**重启电脑**

⑤ 重启后，Docker Desktop 会自动启动，在桌面右下角任务栏可以看到鲸鱼图标 🐳

⑥ 等待图标变为稳定状态（不再闪烁），表示 Docker 已就绪

## 第四章 配置 AI 模型

### 获取 DeepSeek API Key

① 打开浏览器，访问 https://platform.deepseek.com

② 点击注册账号（用手机号或邮箱注册即可）

③ 登录后，点击左侧菜单 **API Keys**

④ 点击 **创建 API Key**，给密钥起个名字（比如"公众号发文"）

⑤ **复制**生成的密钥（格式类似 `sk-xxxxxxxx`），妥善保管

> 注意：密钥只显示一次，请立即复制保存！

【此处插入截图：DeepSeek 平台 API Keys 页面】

### 填写配置

① 在项目文件夹中，找到 `.env` 文件

② 右键 → **打开方式** → **记事本**

③ 找到这一行：
```
AI_API_KEY=your-api-key-here
```

④ 把 `your-api-key-here` **替换成**你刚才复制的密钥，变成：
```
AI_API_KEY=sk-你的真实密钥
```

⑤ 找到这一行：
```
WECHAT_AUTHOR=your-name
```
把 `your-name` 改成你的公众号作者名（比如你的笔名）

⑥ 点击 **文件 → 保存**，关闭记事本

## 第七章 使用 n8n 生成文章

### 首次设置 n8n（仅第一次）

① 打开浏览器，访问 http://localhost:5678

② 第一次打开会提示创建账号：
- 输入你的邮箱
- 输入密码（想一个好记的）
- 点击 **Next** 完成设置

③ 登录后，点击右上角 **Import from File**

④ 选择项目文件夹中的 `workflows` → `wechat_article_auto_publish.json`

⑤ 导入成功后，左侧会出现一个工作流，名字叫 **"WeChat Article Auto-Publish"**

【此处插入截图：n8n 导入工作流界面】

### 生成一篇文章

① 点击左侧的 **"WeChat Article Auto-Publish"** 工作流

② 点击右上角的 **Test workflow** 按钮（或 **Execute workflow**）

③ 在弹出的窗口中输入文章主题，例如：
```
2026年AI办公工具发展趋势
```

④ 点击运行

⑤ 系统会自动执行以下步骤：
- 第 1 步：AI 生成文章内容
- 第 2 步：SEO 优化标题和关键词
- 第 3 步：质量检测 + 内容合规检查
- 第 4 步：自动上传到公众号草稿箱

⑥ 绿色对勾表示全部通过，文章已进入草稿箱。

【此处插入截图：n8n 工作流运行成功界面】

## 第八章 检查公众号草稿并发布

① 回到系统自动打开的浏览器窗口（微信公众号后台）

② 点击左侧菜单 **管理 → 素材管理**

③ 在顶部切换标签到 **草稿箱**

④ 你会看到刚才自动生成的文章草稿

⑤ 点击文章标题打开预览

⑥ 检查以下内容：
- 标题是否正确
- 作者名是否正确
- 正文内容是否完整
- 排版是否正常

⑦ 确认无误后，点击 **发表** 按钮

⑧ 按照提示完成发布

> 最后一步"点击发布"必须由你手动操作。这是微信平台的规定，系统无法代替。

## 第十一章 常见问题

### Q1：需要每天扫码吗？

**不需要。** 只有第一次需要扫码。系统会保存登录状态，以后自动恢复。

### Q2：如果登录失效了怎么办？

删除项目文件夹中的 `data\browser\wechat_clean_profile` 文件夹，然后重新运行 `start.bat`，会重新弹出扫码页面。

### Q3：文章生成后为什么没有自动发布？

个人订阅号不支持自动发布功能。文章会存入**草稿箱**，需要你登录公众号后台，人工预览确认后，点击"发表"。

### Q4：生成失败怎么办？

依次检查：
1. 网络是否正常（能否打开网页）
2. Docker Desktop 是否在运行（右下角有没有鲸鱼图标）
3. `.env` 文件中的 API Key 是否填写正确
4. DeepSeek 账户是否有余额

### Q5：微信提示安全风险怎么办？

如果微信后台提示检测到异常插件或自动化工具，关闭浏览器窗口，运行 `stop.bat`，等几分钟后重新运行 `start.bat` 即可。

### Q6：n8n 网页打不开？

- 等 30 秒再试（n8n 启动需要时间）
- 确认 Docker Desktop 正在运行
- 确认地址输入的是 `http://localhost:5678`

### Q7：可以修改文章风格吗？

可以。在 n8n 工作流中，找到 "Generate Article" 这个节点，把 `style` 的值改为：
- `marketing` → 营销风格
- `science` → 科普风格
- `brand` → 品牌故事风格
