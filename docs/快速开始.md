# 🚀 快速开始 — 10 分钟完成安装

> **适用对象**：非技术人员，无需编程基础，无需 Python 经验。

---

## 安装前准备

| 你需要的东西 | 说明 |
|------------|------|
| 💻 Windows 电脑 | Win10 / Win11 |
| 🔑 DeepSeek API Key | 免费注册即可获取 |
| 🌐 网络连接 | 安装过程需要下载文件 |
| ⏱️ 10 分钟 | 包含等待下载的时间 |

### 获取 DeepSeek API Key（2 分钟）

1. 打开浏览器访问：**https://platform.deepseek.com**
2. 使用手机号或邮箱注册账号
3. 登录后点击左侧菜单「API Keys」
4. 点击「创建 API Key」，复制生成的 Key（格式：`sk-xxxx...`）
5. ⚠️ **妥善保存**，这个 Key 只在创建时显示一次

> 💰 费用说明：DeepSeek 价格很低，生成一篇文章约 ¥0.01-0.05。

---

## 安装步骤

### 第一步：安装 Python（2 分钟）

1. 打开：**https://www.python.org/downloads/**
2. 点击黄色按钮「Download Python 3.12.x」下载
3. 双击下载的安装包
4. ⚠️ **务必勾选底部的「Add Python to PATH」**
5. 点击「Install Now」→ 等待完成 → 关闭窗口

> 验证：按 `Win+R`，输入 `cmd`，输入 `python --version`，看到版本号即成功。

---

### 第二步：解压项目文件夹（1 分钟）

将收到的 `公众号AI运营系统_v2.2.zip` 解压到任意位置。

> 建议解压到桌面或 `D:\`，路径不能包含中文（否则可能出错）。

---

### 第三步：一键安装依赖（3 分钟）

1. 进入解压后的项目文件夹
2. **双击 `install.bat`**
3. 等待自动完成：安装 Python 包 → 下载浏览器引擎 → 创建配置文件
4. 看到「安装完成」提示后按任意键关闭

> 过程中如果杀毒软件弹窗，选择「允许」。这是正常的。

---

### 第四步：配置 API Key（1 分钟）

安装完成后，项目文件夹中会出现 **`.env`** 文件。

用记事本打开 `.env`，修改以下两行：

```
AI_API_KEY=sk-你的DeepSeek密钥
ADMIN_TOKEN=你的管理令牌（自定义，如 mytoken123）
```

保存并关闭。

> 💡 也可以在启动系统后，通过 Web 页面直接配置。

---

### 第五步：启动系统

**双击 `start.bat`**，看到以下信息表示成功：

```
系统已就绪！
工作台：http://localhost:8000
```

浏览器会自动打开工作台页面。

---

## 🎯 生成第一篇文章

### 1. 打开工作台
浏览器访问：**http://localhost:8000**

### 2. 系统配置（仅首次）
点击顶部「⚙️ 系统配置」：
- 填入 DeepSeek API Key
- 填入管理令牌
- 点击「测试连接」确认 Key 有效
- 点击「保存配置」

### 3. 创建文章
- 回到「🏠 首页」
- 输入文章主题，例如：「AI 对普通人工作的影响」
- 选择文章风格：科技 / 商业 / 知识科普 / 个人随笔
- 选择文章长度：短篇 / 中篇 / 长篇
- 点击「生成文章」

### 4. 查看结果
生成完成后，展示：
- 📝 文章标题 + 正文
- 🖼️ AI 配图（如已开启）
- 📊 质量评分
- 🎨 排版预览

### 5. 发布到微信草稿
- 点击「发布到微信草稿」
- 首次使用需要扫码登录微信公众号
- 登录后自动保存到草稿箱
- 打开微信公众号后台 → 素材管理 → 草稿箱，即可查看

## 🎛️ 可选配置

| 功能 | 配置方法 |
|------|---------|
| 🖼️ AI 配图 | .env 中设置 `IMAGE_API_KEY`，开启 `ENABLE_IMAGE=true` |
| 📚 知识库增强 | .env 中设置 `OBSIDIAN_PATH` 为你的笔记文件夹路径 |
| 📋 纯文本模式 | .env 中设置 `ENABLE_IMAGE=false`，不生成配图也正常使用 |
