# 微信公众号 AI 自动运营系统 — 功能说明

> **版本**：v2.1 | **日期**：2026-07-14

---

## 系统概述

一套面向微信公众号个人订阅号的 AI 内容创作工具。输入一个主题或一张图片，系统自动完成文章生成、排版优化，并上传至公众号草稿箱，您只需登录后台确认发布即可。

---

## 部署架构（一张图看懂）

```
您的电脑（Windows 10/11）
│
├── n8n（Docker 容器）
│   └── 工作流编排调度
│       定时触发 / 手动触发 / Webhook 触发
│       │
│       └── HTTP 调用 ──────────────────────┐
│                                            │
├── FastAPI 后端（Python 进程）  ←──────────┘
│   │  端口 8000
│   │  提供 18 个 REST API 端点
│   │
│   ├── AI 文章生成（调用 DeepSeek API）
│   ├── 图片生成（调用即梦 AI API）
│   ├── 视觉理解（调用通义千问 VL API）
│   ├── 排版引擎（本地 Python）
│   ├── 质检系统（本地规则 + AI）
│   │
│   └── Playwright 浏览器自动化
│        └── 操作微信公众号后台 → 上传草稿箱
│
├── Web 操作面板（浏览器打开 localhost:8000）
│   └── 可视化操作，不依赖 n8n
│
└── 本地文件
    ├── storage/articles.db（SQLite 数据库）
    ├── prompts/*.txt（AI 提示词模板）
    └── wechat_browser_profile（微信登录态）
```

**一句话总结**：n8n 是"遥控器"，FastAPI 是"发动机"，你可以在遥控器上编流程，也可以直接操作发动机。

## 接入方式详解

### 方式一：Web 操作面板（开箱即用）

> **不需要 n8n，不需要写代码。双击 start.bat 即用。**

浏览器打开 `http://localhost:8000` → 可视化界面操作：

- 输入主题 → 选风格 → 点击"生成并发布"
- 或粘贴图片链接 → AI 视觉理解 → 生成文章
- 或粘贴公众号/抖音链接 → 竞品分析 → 生成文章

进度条实时显示当前步骤，结果直接展示。

---

### 方式二：接入 n8n 工作流（自动化编排）

> **n8n 通过 Docker 运行，已经预配好工作流 JSON 文件，导入即用。**

#### n8n 是什么
n8n 是一个可视化工作流编排工具，你可以像画流程图一样编排"什么时候 → 做什么"。

#### 已提供的工作流

| 工作流文件 | 功能 | 触发方式 |
|-----------|------|---------|
| `workflows/wechat_article_auto_publish.json` | v1 基础流程 | n8n 手动触发 / Webhook |
| `workflows/wechat_article_v2_pipeline.json` | v2 完整流程（含配图+排版） | n8n 手动触发 |

#### 工作流内部原理

n8n 本身不做 AI、不排版、不操作浏览器。它的工作是**按顺序调用 FastAPI 接口**：

```
n8n 工作流节点                    FastAPI 接口
──────────────────────────────────────────────
[Manual Trigger]                    (你点击"运行")
       │
[生成文章]     ──HTTP POST──→  POST /api/article/generate
       │                       AI 生成文章，返回标题+正文+摘要
[SEO优化]      ──HTTP POST──→  POST /api/article/seo
       │                       优化标题和关键词
[质量检测]     ──HTTP POST──→  POST /api/article/check
       │                       评分 + 合规检测
[质量门判断]   ──IF 判断──→    score≥70 → 继续；否则停止
       │
[发布草稿]     ──HTTP POST──→  POST /api/publish/draft
                               Playwright 自动上传微信草稿箱
[完成]
```

#### n8n 扩展能力（你可以自己加的）

n8n 支持 400+ 集成节点，你可以自行扩展：
- **定时发布**：加一个 Schedule Trigger，设定每天早上 8 点自动运行
- **多平台通知**：加钉钉/飞书/邮件节点，发布成功/失败时通知你
- **RSS 抓取**：加 RSS 节点自动获取热点 → 作为文章主题
- **条件分支**：根据质检分数决定要不要人工审核

---

### 方式三：直接调用 API（对接你自己的系统）

> **如果你有自己的后台系统或脚本，可以直接调 API。**

所有 API 不需要认证（本地部署，不暴露公网）。

#### 常用 API 速查

**生成文章**：
```bash
curl -X POST http://localhost:8000/api/article/generate \
  -H "Content-Type: application/json" \
  -d '{"topic": "AI时代", "style": "marketing", "author": "你的名字"}'
```

**图片驱动创作（v2.1）**：
```bash
curl -X POST http://localhost:8000/api/v2/image-article/generate \
  -H "Content-Type: application/json" \
  -d '{"image_url": "https://你的图片地址.jpg", "style": "marketing"}'
```

**完整 v2 流程（含配图+排版）**：
```bash
curl -X POST http://localhost:8000/api/v2/pipeline/run \
  -H "Content-Type: application/json" \
  -d '{"topic": "AI创业趋势", "template": "technology"}'
```

**查看任务历史**：
```bash
curl http://localhost:8000/api/tasks?limit=20
```

完整 API 文档：启动系统后浏览器打开 `http://localhost:8000/docs`（Swagger 交互文档）

## 三种创作模式

### 📝 模式一：主题创作
> 输入一个主题 → AI 自动写文 → 排版 → 草稿箱

**Web 面板操作**：输入主题 → 选风格 → 点击按钮
**n8n 操作**：运行 `wechat_article_auto_publish` 工作流，输入 `{"topic": "主题"}`
**API 调用**：`POST /api/article/generate` → `POST /api/publish/draft`

---

### 🖼️ 模式二：图片驱动创作（v2.1 新增）
> 上传一张图片 → AI 理解图片 → 围绕图片写文 → 草稿箱

**Web 面板操作**：粘贴图片链接 → 选风格 → 点击按钮
**n8n 操作**：需自行添加 HTTP Request 节点调用 `/api/v2/image-article/generate`
**API 调用**：`POST /api/v2/image-article/generate`

---

### 🔗 模式三：竞品分析创作
> 粘贴公众号/抖音链接 → AI 分析 → 新文章 → 草稿箱

**Web 面板操作**：粘贴链接 → 选风格 → 点击按钮
**n8n 操作**：需自行添加 HTTP Request 节点调用 `/api/article/generate_from_source`
**API 调用**：`POST /api/article/generate_from_source`

## 内容能力

### ✍️ 4 种写作风格

| 风格 | 特点 | 适用 |
|------|------|------|
| 营销推广 | 痛点引入 + CTA 结尾 | 产品推广、活动宣传 |
| 科技科普 | 通俗易懂，分层清晰 | 技术解读、知识分享 |
| 案例分析 | 深度解读，数据支撑 | 行业分析、案例拆解 |
| 品牌故事 | 有温度的叙事 | 品牌宣传、创始人故事 |

### 🎨 4 种排版模板

| 模板 | 配色 | 适用 |
|------|------|------|
| 科技前沿 | 深色标题，简洁现代 | 科技/AI/技术类公众号 |
| 商务专业 | 蓝色标题，稳重大气 | 企业公众号 |
| 知识科普 | 分层明显，阅读友好 | 教育/知识类公众号 |
| 个人 IP | 温暖色调，故事感 | 个人博主 |

### 🔍 智能增强

- **小标题**：自动检测"首先/其次/第一、"等 → 转为醒目标题样式
- **数据加粗**：自动检测"30%"、"500万"等 → 红色加粗
- **引用美化**：自动检测"据统计/数据显示"等 → 灰底引用块

### ✅ 内容质检

- 标题质量检测（5-64 字）
- 正文完整性检测（≥200 字）
- 结构检测（有无小标题分段）
- 合规检测（敏感词 + 广告法极限词）

## 部署安装（5 步）

### 第一步：安装 Python 3.12+
https://www.python.org/downloads/ （安装时勾选"Add Python to PATH"）

### 第二步：安装 Docker Desktop
https://www.docker.com/products/docker-desktop/ （n8n 的运行环境）

### 第三步：获取 API Key
- **DeepSeek**（必选）：https://platform.deepseek.com → 获取 API Key，月费约 ¥10-50
- **即梦 AI**（可选，配图功能需要）：火山引擎控制台获取
- **通义千问 VL**（可选，图片理解功能需要）

### 第四步：配置
编辑 `.env` 文件，填入你的 API Key：
```
AI_API_KEY=sk-xxxxx           # DeepSeek
IMAGE_API_KEY=xxxxx            # 即梦（可选）
VISION_API_KEY=xxxxx           # 通义千问 VL（可选）
WECHAT_AUTHOR=你的公众号名字
```

### 第五步：启动
双击 `start.bat` → 等待 10-20 秒 → 浏览器弹出微信扫码页 → 手机扫码登录

## 不做什么

- ❌ 不自动群发（仅上传草稿箱，你人工确认发布）
- ❌ 不需要公众号官方 API（个人订阅号不支持，系统通过浏览器自动化完成）
- ❌ 不存储你的微信密码（登录通过手机扫码）
- ❌ 不需要服务器（全部在你自己的电脑上运行）
