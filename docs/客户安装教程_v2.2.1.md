## 二、安装 Python

Python 是本系统运行的基础环境，必须安装。

### 操作步骤

1. 打开浏览器，访问：**https://www.python.org/downloads/**
2. 点击页面中央的黄色大按钮 **Download Python 3.12.x** 下载安装包
3. 双击下载好的文件（文件名类似 python-3.12.x-amd64.exe）
4. **重要！在安装界面的最下方，一定要勾选 ☑️「Add Python to PATH」**
5. 点击 **Install Now** 按钮，等待进度条跑完
6. 点击 **Close** 完成安装

### 验证是否安装成功

1. 按键盘上的 **Win 键 + R**，输入 cmd，回车
2. 在弹出的黑色窗口中输入：

`
python --version
`

3. 如果显示 Python 3.12.x，说明安装成功！

> 如果提示 "不是内部或外部命令"，说明没勾选 "Add Python to PATH"，请重新安装并确保勾选该项。

## 三、安装 Docker Desktop（可选）

> Docker 用于运行自动化工作流引擎。如果你只需要「AI写文章 + 手动发布」功能，**可以不装 Docker**，跳过本章即可。

### 操作步骤

1. 打开浏览器，访问：**https://www.docker.com/products/docker-desktop/**
2. 点击 **Download Docker Desktop for Windows** 下载
3. 双击安装包，一路点 "下一步" 完成安装
4. **安装完成后，重启电脑**
5. 重启后，桌面右下角任务栏会出现一只鲸鱼图标 🐳
6. 等待鲸鱼图标不再闪烁，变为稳定状态，说明 Docker 已就绪

### 验证

在命令行（Win+R → cmd）中输入：

`
docker --version
`

看到版本号即表示安装成功。

## 四、获取 API 密钥

系统需要两个 API 密钥来工作。**这是付费服务，但费用很低。**

### 4.1 DeepSeek API Key（必填 — AI 写文章）

DeepSeek 是国内的 AI 大模型，用来生成文章内容。

**获取步骤：**

1. 打开浏览器，访问：**https://platform.deepseek.com/**
2. 点击右上角「注册」，用手机号或邮箱注册账号
3. 登录后，点击左侧菜单「API Keys」
4. 点击「创建 API Key」，随便起个名字（比如"公众号系统"）
5. **复制生成的密钥**（格式为 sk- 开头的一长串字母数字）
6. 保存好这个密钥，后面会用到

> DeepSeek 费用很低，约 1 元/百万字。新用户通常有免费额度。

### 4.2 火山方舟 API Key（可选 — AI 生成插图）

火山方舟（字节跳动旗下）提供 AI 图片生成服务。

**获取步骤：**

1. 打开浏览器，访问：**https://console.volcengine.com/ark/**
2. 注册/登录火山引擎账号
3. 进入「模型广场」，找到 **Seedream 4.0** 模型
4. 点击「开通服务」，按提示完成
5. 进入「API Key 管理」，创建一个 API Key
6. **复制密钥**（格式为 rk- 开头）

> 如果不需要 AI 自动生成文章配图，可以不配置此项。

## 五、安装项目文件

### 5.1 解压项目

1. 找到我们发给你的项目压缩包（格式为 .zip）
2. 右键压缩包 → 选择 **「解压到当前文件夹」**
3. 将解压出来的文件夹放到一个方便的位置，例如：

`
D:\公众号AI运营系统
`

> 路径不要有中文，建议用英文或拼音。

4. 打开文件夹，确认里面有这些文件：
   - 一键安装.bat
   - 一键启动.bat
   - 一键停止.bat
   - pp 文件夹
   - docs 文件夹

### 5.2 一键安装

1. **双击 一键安装.bat**
2. 黑色窗口会自动运行，包含以下步骤：
   - 检查 Python 环境
   - 安装 Python 依赖包
   - 安装浏览器引擎（用于自动发布到公众号）
   - 创建配置文件
   - 检查 Docker（可选）
3. 等待出现 **"安装完成！"** 提示后，按任意键关闭窗口

> 如果安装过程中报错，请截图发给我们。

## 六、配置系统

这是最关键的步骤 — 把你的 API 密钥填入配置文件。

### 6.1 打开配置文件

1. 在项目文件夹中，进入 pp 子文件夹
2. 找到 .env 文件（注意：文件名就是 .env，没有扩展名）
3. 右键 .env → 选择 **「打开方式」→「记事本」**

### 6.2 填写 API 密钥

在记事本中找到以下内容并修改：

`
# DeepSeek API（必填）
AI_API_KEY=sk-你的deepseek密钥填在这里
AI_BASE_URL=https://api.deepseek.com
AI_MODEL=deepseek-chat

# 图片生成（可选，不需要可保持注释）
IMAGE_PROVIDER=jimeng
IMAGE_API_KEY=ark-你的火山方舟密钥填在这里
IMAGE_BASE_URL=https://ark.cn-beijing.volces.com/api/v3
IMAGE_MODEL=doubao-seedream-4-0-250828
`

**只需要修改两行：**
- AI_API_KEY= 后面填入你的 DeepSeek 密钥
- IMAGE_API_KEY= 后面填入你的火山方舟密钥（不需要生图就跳过）

其他设置保持默认即可：

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| ADMIN_TOKEN | wxai2026! | 管理密码，可自行修改 |
| PUBLISH_MODE | uto | 发布模式：auto=自动保存草稿 |
| CLIENT_MODE | 	rue | 客户模式（隐藏技术界面） |

### 6.3 保存文件

点击记事本的 **「文件」→「保存」**，然后关闭记事本。

## 七、启动系统

### 7.1 启动

1. **双击 一键启动.bat**
2. 黑色窗口会出现，显示启动信息
3. 等待约 5 秒，浏览器会自动打开系统界面
4. 如果没有自动打开，手动在浏览器输入：**http://localhost:8000**

### 7.2 系统界面

打开后你会看到以下功能页面：

| 标签页 | 功能 |
|--------|------|
| **写文章** | 输入主题，AI 自动生成文章 |
| **文章列表** | 查看已生成的所有文章 |
| **发布管理** | 管理发布任务 |
| **系统状态** | 查看系统运行状态 |

### 7.3 关闭系统

- 双击 一键停止.bat 即可关闭
- 或者直接关闭黑色窗口

> 关闭后，已生成的文章不会丢失，存储在 pp/storage/articles.db 中。

## 八、日常使用

### 8.1 写一篇文章

1. 启动系统后，点击顶部的 **「写文章」** 标签
2. 在输入框中填写文章主题（例如："如何选择适合自己的运动鞋"）
3. 选择文章风格：营销推广 / 知识科普 / 案例拆解 / 品牌故事
4. 选择文章长度：短篇 / 中篇 / 长篇
5. 填写公众号作者名
6. 可选：开启「AI生图」自动生成配图
7. 点击 **「开始生成」** 按钮
8. 等待几十秒，文章生成后会在右侧预览

### 8.2 发布到公众号

1. 在文章列表中，找到生成好的文章
2. 点击 **「发布到草稿箱」**
3. 系统会自动打开公众号后台，填入标题、正文、摘要
4. 自动保存到草稿箱
5. 你只需登录公众号后台，在草稿箱中检查并手动群发

> 首次发布需要扫码登录公众号，之后系统会记住登录状态。

### 8.3 注意事项

- 使用前确保电脑**不关机、不休眠**
- 发布时保持浏览器窗口可见，不要最小化
- 如果长时间不用，DeepSeek 密钥可能过期，需要重新生成

## 九、常见问题

### Q1：提示「AI API 调用失败」

**原因**：DeepSeek API 密钥配置错误或已过期。

**解决**：
1. 登录 https://platform.deepseek.com/
2. 进入 API Keys 页面
3. 创建新的 API Key
4. 复制密钥，更新到 pp\.env 文件中
5. 重启系统

### Q2：文章中有奇怪的代码符号

已修复。如果还出现，重启系统后重试即可。

### Q3：Docker 启动失败

**现象**：Docker Desktop 无法启动，或提示 WSL 错误。

**解决**：
1. 确认电脑开启了虚拟化（在 BIOS 中开启 VT-x/AMD-V）
2. 在 PowerShell（管理员）中运行：wsl --install
3. 重启电脑后再试

### Q4：图片生成失败

**原因**：火山方舟的 Seedream 模型需要开通服务。

**解决**：
1. 登录 https://console.volcengine.com/ark/
2. 进入「模型广场」，找到 Seedream 4.0
3. 确认已「开通服务」
4. 确认账户有余额（或体验额度）

### Q5：端口被占用

**现象**：启动时提示端口 8000 已被占用。

**解决**：运行 一键停止.bat 后再启动。如果还不行，重启电脑。

### Q6：如何备份数据？

复制整个项目文件夹到安全的位置即可。数据库文件在 pp/storage/ 下。
