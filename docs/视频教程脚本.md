# 视频教程脚本（5分钟）

> 面向普通客户的演示视频，展示从零到生成第一篇文章的全过程。

---

## 录制准备

- 电脑已安装好 Python、Docker Desktop
- 已获取 DeepSeek API Key
- 已解压项目文件
- 准备一个示例主题："2026年AI办公工具发展趋势"
- 确保网络畅通

- 所有输入框、按钮、关键文字用**红框或高亮**标注

- 语速适中，每条讲解词不超过15秒
