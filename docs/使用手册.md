# 公众号AI智能运营系统 V2.2 — 使用手册

## 一、启动系统

1. 双击「一键启动.bat」
2. 等待约5秒，浏览器自动打开
3. 或手动访问：http://localhost:8000

## 二、生成文章

1. 在「文章生成」页输入主题
2. 选择风格（营销/科普/案例/品牌）
3. 选择长度（短/中/长）
4. 填写公众号作者名
5. 可选：开启「AI生图」
6. 点击「开始生成」，等待30-60秒

## 三、发布到公众号

1. 首次使用需扫码登录公众号
2. 系统自动填入标题、正文、摘要到编辑器
3. 自动保存到草稿箱
4. 您需手动登录公众号后台群发

## 四、系统配置

在「系统配置」页可修改API密钥，保存后立即生效。

## 五、关闭系统

双击「一键停止.bat」或关闭命令行窗口。

## 六、常见问题

**API调用失败**：检查DeepSeek余额 / 测试连接 / 检查网络
**登录失败**：确保Chrome已装 / 手机微信已登录公众号
**图片生成失败**：确认火山方舟已开通Seedream 4.0模型
