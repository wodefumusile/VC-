# 快速使用说明（一页速查卡）

> 打印出来贴在显示器旁边，每天照着做。

---

## 启动（每天做一次）

```
双击 start.bat → 等待黑色窗口显示"系统启动成功"
```

- 启动前确认右下角有 Docker 鲸鱼图标 🐳
- 不要关闭黑色窗口

---

## 生成文章

```
1. 打开浏览器 → http://localhost:5678
2. 点击"Workflows" → "WeChat Article Auto-Publish"
3. 点击 "Test workflow"
4. 输入文章主题，回车
5. 等待执行完成（约 30-60 秒）
```

---

## 审核发布

```
1. 系统浏览器自动打开微信公众号后台
2. 左侧菜单 → 管理 → 素材管理 → 草稿箱
3. 点击文章预览
4. 检查标题、作者、正文
5. 点击"发表"
```

---

## 关闭（用完就关）

```
双击 stop.bat → 看到"系统已关闭"
```
