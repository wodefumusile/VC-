# 🔧 故障排查指南

> 遇到问题请按顺序检查，90% 的问题可以通过以下步骤解决。

---

## 安装问题

### ❌ 双击 install.bat 提示「未检测到 Python」

**原因**：Python 未安装或未添加到系统路径。

**解决**：
1. 重新安装 Python 3.12+：https://www.python.org/downloads/
2. 安装时 **必须勾选**「Add Python to PATH」
3. 安装完成后 **重启电脑**
4. 再次双击 `install.bat`

---

### ❌ pip install 报错「connection timeout」

**原因**：网络连接 PyPI 官方源不稳定。

**解决**：使用国内镜像源安装

打开命令提示符（`Win+R` → 输入 `cmd`），进入项目文件夹后执行：

```
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

然后再执行：
```
playwright install chromium
```

---

### ❌ playwright install 卡住不动

**原因**：Chromium 浏览器包约 150MB，下载需要时间。

**解决**：
1. 耐心等待 3-5 分钟
2. 确保网络稳定
3. 如多次失败，手动执行：`playwright install chromium`

---

### ❌ 杀毒软件报毒/拦截

**原因**：Playwright 的 Chromium 浏览器可能被误报。

**解决**：
1. 这是正常现象（Playwright 是微软官方工具）
2. 在杀毒软件中选择「允许」或「添加信任」
3. 重新运行 `install.bat`

---

## 启动问题

### ❌ 双击 start.bat 后窗口一闪而过

**原因**：Python 依赖未安装或 .env 配置有问题。

**解决**：
1. 先运行 `install.bat` 确保依赖完整
2. 检查 .env 文件是否存在
3. 在项目文件夹地址栏输入 `cmd`，手动运行：
   ```
   python -m backend.main
   ```
   查看具体错误信息

---

### ❌ 提示「端口 8000 已被占用」

**原因**：系统已在运行，或其他程序占用了 8000 端口。

**解决**：
1. 检查是否已经打开过系统
2. 运行 `stop.bat` 关闭已有进程
3. 或者修改 .env 中的 `SERVER_PORT` 为其他端口（如 8001）

---

### ❌ 浏览器打开 http://localhost:8000 显示空白

**原因**：服务还未完全启动。

**解决**：
1. 等待 5-10 秒后刷新页面
2. 检查命令行窗口是否有红色报错
3. 访问 http://localhost:8000/health 确认服务状态

---

## 使用问题

### ❌ 生成文章时报错「API Key 无效」

**原因**：DeepSeek API Key 未配置或已过期。

**解决**：
1. 登录 https://platform.deepseek.com 检查 Key 状态
2. 在「⚙️ 系统配置」页面重新填入 Key
3. 点击「测试连接」确认 Key 有效
4. 注意 Key 格式为 `sk-` 开头

---

### ❌ AI 配图不显示 / 报错

**原因**：图片 API Key 未配置或额度不足。

**解决**：
1. 如果不需要配图，在配置中将 `ENABLE_IMAGE` 设为 `false`
2. 系统会自动跳过图片生成，文章仍可正常发布
3. 如需配图，检查 `IMAGE_API_KEY` 是否正确

---

### ❌ 发布微信草稿时扫码失败

**原因**：微信登录态过期或浏览器兼容问题。

**解决**：
1. 浏览器窗口会自动打开微信公众号登录页
2. 使用手机微信扫描屏幕上的二维码
3. 在手机上点击「确认登录」
4. 如扫码无反应，刷新页面重试
5. 登录成功后后续自动使用，无需重复扫码

---

### ❌ 加载 Obsidian 知识库很慢

**原因**：笔记文件较多时首次扫描需要时间。

**解决**：
1. 首次扫描 1000 个文件约 10 秒
2. 之后会使用缓存，仅扫描新增/修改的文件
3. 如不需要知识库功能，将 `OBSIDIAN_PATH` 留空

---

### ❌ 文章质量不理想

**原因**：提示词或模型参数需要调整。

**解决**：
1. 尝试更具体、更有指向性的主题描述
2. 选择更适合的文章风格
3. 开启知识库功能，让 AI 参考你的个人笔记
4. 调整文章长度参数

---

## 数据与安全

### ❌ 如何备份数据？

**系统自动备份**：数据库每天自动备份到 `backup/` 文件夹。

**手动备份**：复制以下文件夹即可：
- `storage/` — 数据库文件
- `.env` — 配置文件

---

### ❌ API Key 会被泄露吗？

1. API Key 存储在本地 `.env` 文件中
2. Web 页面展示时会自动脱敏（显示为 `sk-abc****xyz`）
3. 修改配置需要管理员令牌验证
4. **不要将 .env 文件分享给他人**

---

## 仍需帮助？

1. 查看 `logs/app.log` 和 `logs/error.log` 了解详细错误
2. 记录错误信息，联系技术支持

---

> 💡 80% 的问题都可以通过「重新启动系统」解决。先运行 `stop.bat`，再运行 `start.bat`。
